# 🍔 Smart Food Order System

A responsive **Food Ordering Web Application** built with **Flask** and **SQLite**. Users can register, log in, browse the food menu, add items to the cart, place orders, and view their order history.

## ✨ Features
- 🔐 User Authentication
- 🍕 Food Menu
- 🛒 Shopping Cart
- 💳 Order Placement
- 📦 Order History
- 📱 Responsive Design

## 🛠️ Tech Stack
- 🐍 Python
- 🌐 Flask
- 🗄️ SQLite
- 🎨 HTML, CSS, JavaScript
- 🅱️ Bootstrap 5

## 🚀 Run the Project

```bash
pip install -r requirements.txt
python app.py
```
